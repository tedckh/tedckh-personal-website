# Personal Portfolio of Ted Chiu Ka Ho

This repository contains the source code for my personal portfolio website, built from the ground up with Next.js and a modern, interactive tech stack.

**Live Site:** [**https://tedckh.com**](https://tedckh.com)

---

## ✨ Features

*   A curated and paginated list of 15 professional projects.
*   A robust, persistent Light/Dark theme switcher that defaults to the user's system preference.
*   A clean, professional, and fully responsive design.
*   Smooth animations on scroll and hover, powered by Framer Motion.
*   An interactive image gallery modal for project screenshots.
*   A reusable and scalable component architecture.

## 🛠️ Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) (React)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **Styling:** [styled-components](https://styled-components.com/)
*   **Animation:** [Framer Motion](https://www.framer.com/motion/)
*   **Data Fetching:** [TanStack Query](https://tanstack.com/query/latest)
*   **Deployment:** [Vercel](https://vercel.com/)

## 🚀 Running Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/tedckh/tedckh-personal-website.git
    cd tedckh-personal-website
    ```

2.  **Install dependencies:**
    This project uses `pnpm` as its package manager.
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    To run with the stable Webpack bundler (recommended):
    ```bash
    pnpm run dev:webpack
    ```
    Or to run with the default Turbopack server:
    ```bash
    pnpm dev
    ```

4.  **Open the application:**
    Open [http://localhost:3000](http://localhost:3000) in your browser.
